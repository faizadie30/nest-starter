# About

Starter Pack Nextjs

# Tech Stack

1. dotenv
2. postgresql
3. typescript
4. nestjs
5. validator and transformer validator
6. typeorm
7. yarn
8. nodejs version: 18.15.0
9. redis
10. socket.io
11. passportjs
12. jwt
13. create token and verify with manual logic

# How To Start

1. install

```ssh
yarn install
```

2. running in development

```ssh
yarn stat:dev
```

3. CLI Nest

```ssh
nest
```
